# School Management System - Backend API

Complete backend API for School Management System with role-based access control.

## Features

- 🔐 JWT Authentication & Authorization
- 👥 Multi-role system (Admin, Teacher, Student, Parent)
- 📚 Class & Subject Management
- 👨‍🏫 Teacher Assignment & Management
- 📊 Attendance Tracking
- 📝 Result Management with Approval Workflow
- 💰 Fee Management with Receipt Generation
- 📢 Announcements with File Uploads (Cloudinary)
- 🗓️ Timetable Management
- 📄 PDF Generation for Results & Receipts

## Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT
- **File Upload**: Multer + Cloudinary
- **PDF Generation**: PDFKit
